YouDrive - Il diario intelligente del tuo veicolo

Quante volte capita, a chi possiede un veicolo o ne gestisce diversi, di dimenticare una scadenza importante, un pagamento, un tagliando o una manutenzione programmata?
YouDrive nasce proprio per eliminare questo problema: tu guidi, noi pensiamo al resto.
YouDrive è un diario digitale del veicolo, progettato per aiutare proprietari, famiglie e aziende a tenere sotto controllo stato, impiego, scadenze e panoramica generle dei propri veicoli.
L'applicazione registra, organizza e ricorda automaticamente tutte le attività essenziali, trasformando la gestione del veicolo in un processo semplice, intuitivo e senza stress.

Linguaggi utilizzati

  • HTML
  • CSS
  • JAVA
  • SQL
  • JAVASCRIPT
  • TYPESCRIPT
  • API

Funzionalità

  • Dashboard veicolo con panoramica immediata di scadenze e stato del veicolo
  • Promemoria intelligenti per bollo, assicurazione, revisione, tagliandi
  • Storico manutenzioni con costi, date, chilometri e note
  • Gestione multipli veicoli
  • Prenotazione check meccanico direttamente dall'app
